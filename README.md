Expense Tracker
Автор:
Сорокин Никита Андреевич

Дата: 05.05.2026

Описание:

Простое приложение на Python для учёта расходов.
Позволяет добавлять, фильтровать и считать сумму расходов.

Запуск:
```bash
python expense_tracker.py
```
Использование

Введите сумму, категорию и дату (YYYY-MM-DD)

Нажмите "Добавить расход"

Используйте фильтр при необходимости

Нажмите "Подсчитать"

Данные 
сохраняются в файл expenses.json

Пример:

500, еда, 2026-05-01
120, транспорт, 2026-05-02
